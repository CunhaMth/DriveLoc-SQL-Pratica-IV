# Experiência Prática IV - DriveLoc (Locadora de Veículos)

## Descrição
Projeto de Banco de Dados para gerenciamento de locadora de veículos. Inclui DDL para criação do esquema, DML para inserção de dados, consultas avançadas e operações de manutenção.

## Estrutura do Projeto
- **01_Criacao_Banco_DriveLoc.sql**: Cria o banco, tabelas, chaves primárias e estrangeiras.
- **02_Insert_Dados_DriveLoc.sql**: Insere dados de exemplo (filiais, clientes, veículos, etc.).
- **03_Selects_Avancados_DriveLoc.sql**: Consultas SQL com JOIN, GROUP BY, WHERE, ORDER BY, LIMIT.
- **04_Update_e_Delete_DriveLoc.sql**: Operações de atualização e exclusão com condições.

## Como Executar
1. Abra o MySQL Workbench ou pgAdmin.
2. Execute os scripts na ordem: 01 → 02 → 03 → 04.
3. Verifique os resultados nas consultas.

## Tecnologias
- MySQL 8.0+
- SQL Padrão ANSI
